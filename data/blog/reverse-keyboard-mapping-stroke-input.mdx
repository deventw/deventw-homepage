## 反轉 Keyboard Mapping 步驟

### Step 1: 安裝 Karabiner-Elements
- 打開終端機，執行以下命令安裝 Homebrew（如尚未安裝）：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- 使用 Homebrew 安裝 Karabiner-Elements：

```bash
brew install --cask karabiner-elements
```

### Step 2: 啟用系統的輸入監控
- 安裝後，前往設定 -> 隱私 -> 輸入監控 (Input Monitoring)，開啟權限。
- 在設備中啟用監控。
- 如果是 MacBook，會有 Apple Internal Keyboard。

### Step 3: 啟用 Apple 筆畫輸入法
- 注意 `Stroke-Cantonese` 和 `Stroke-Traditional` 的區別，本文用 `Traditional` 作爲示範。
- 使用 `Cantonese` 時，需要到 Karabiner Event Monitor 查看輸入代碼名稱。

### Step 4: 開啟配置資料夾
- 在 Karabiner-Elements 前往 `misc`，打開配置資料夾。

### Step 5: 編輯 JSON 檔案
- 在 `assets -> complex_modifications` 中打開 JSON 檔案，名稱不重要。
- 我使用 Sublime Text 來編輯 JSON。

### Step 6: 添加和啟用規則
- 在 `complex modifications` 中選擇 `Add Rule`。
- 確認啟用所有六個規則。
- 完成後，就可以使用反轉的筆畫輸入法。
